IRT Curve (3PL Model)

Adjust the parameters below to see how they influence the probability of endorsing an item across different levels of a latent trait (θ).

In Classical Test Theory, every item is treated the same — a correct answer is worth one point regardless of who answered or how hard the question was. Item Response Theory (IRT) replaces this with a probabilistic model: each item has its own characteristic curve that describes the probability of a correct (or endorsed) response as a function of an underlying latent trait, θ.

The three-parameter logistic (3PL) model is the most general of the standard unidimensional IRT models. It describes each item with three parameters: discrimination (a) controls the steepness of the curve — how sharply the item distinguishes between respondents near the difficulty threshold; difficulty (b) locates the curve along the trait axis — higher values mean more of the trait is needed for a correct response; and pseudo-guessing (c) sets a lower asymptote, acknowledging that even very low-ability respondents may endorse the item by chance. Setting c = 0 recovers the 2PL model, and further fixing a to be equal across items gives the Rasch (1PL) model.

Discrimination (a): How well the item differentiates individuals. 1.00
Difficulty (b): The inflection point of the curve.
P(θ=b) = (1+c)/2 — midway between the guessing floor c and 1. 0.00
Guessing/Pseudo-guessing (c): The lower asymptote. 0.00
Try it: Set c = 0 and increase a — the curve steepens into a near-step function, meaning the item sharply discriminates around the difficulty point. Now raise c to 0.25 and notice the curve never reaches 0: even very low-ability respondents have a 25% chance of a correct response (guessing floor). Move b to see how difficulty shifts the inflection point along the trait axis.
The 3PL model assumes unidimensionality (a single latent trait drives responses) and local independence (items are independent given θ). When c = 0 this reduces to the 2PL; when a is also fixed across items, it reduces to the Rasch (1PL) model. In practice, the c parameter is difficult to estimate reliably and is often constrained or set to zero.
